# 🎮 Análise de Dados Steam API

Este projeto é um **Sistema Web** que utiliza a API oficial da Steam para permitir a **análise comparativa de preços e informações de jogos**. O objetivo é fornecer ao usuário dados históricos e comparativos entre diferentes períodos, facilitando a tomada de decisões informadas na hora de comprar jogos.

🔗 **Link do Projeto:** [https://steamapi.kesug.com](https://steamapi.kesug.com)

## 💡 Funcionalidades

O sistema oferece as seguintes funcionalidades principais para auxiliar na análise e compra de jogos:

* **Navegação Fluida:** Interface organizada para fácil transição entre as diferentes telas de análise.
* **Histórico de Pesquisa:** Todas as pesquisas realizadas são salvas em um banco de dados para acesso rápido.
* **Comparação de Períodos:** Permite comparar preços e dados de jogos em diferentes janelas de tempo.
* **Filtro Avançado:** Capacidade de aplicar filtros por categorias para refinar os resultados da busca.
* **Integração Direta:** Links e dados integrados diretamente com a loja da Steam.

## 🛠️ Tecnologias

Este projeto foi construído utilizando as seguintes tecnologias:

* **Backend & Banco de Dados:**
    * **PHP:** Utilizado para a comunicação com o banco de dados e lógica de servidor.
    * **MySQL:** Banco de dados usado para salvar o histórico de pesquisas recentes dos usuários.
* **Frontend & Interatividade:**
    * **HTML & CSS:** Estrutura e estilização da parte visual e de apresentação.
    * **JavaScript (JS):** Responsável pela lógica de requisições assíncronas (AJAX) e animações, garantindo uma usabilidade mais fluida.
* **Infraestrutura:**
    * **API da Steam:** Fonte de dados para todas as análises de jogos.
    * **Hospedagem:** Sistema hospedado em ambiente web para facilitar a usabilidade e acesso.

---

## 🎯 Uso

O sistema tem como foco principal **ajudar na tomada de decisões de compra** de jogos online.

Ao permitir uma **pesquisa fácil e direta** com a plataforma da Steam e oferecer a capacidade de fazer **comparações entre períodos**, o projeto ajuda o usuário a:

* Refletir se o preço atual é o ideal.
* Planejar compras futuras, avaliando se vale a pena aguardar uma possível queda de preço ou promoção.
* Ter acesso a uma **usabilidade mais fluida** por ser um sistema WEB dedicado à análise de preços.
